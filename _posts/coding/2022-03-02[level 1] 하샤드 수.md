##  python 풀이 


```ruby
def solution(x):
    answer=0
    answer1=True
    for i in str(x):
        answer+=int(i)
    
    if x%answer ==0:
        answer1=True
    elif x%answer !=0:
        answer1=False
    
    return answer1

```

간단하게 푸는 법
```ruby
def solution(n): 
  return n % sum([int(c) for c in str(n)]) == 0  
```
