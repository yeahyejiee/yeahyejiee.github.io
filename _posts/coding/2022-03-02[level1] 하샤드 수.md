##  python 풀이 


```ruby

def solution(x):
    answer=0
    answer1=True
    for i in str(x):
        answer+=int(i)

    if x%answer ==0:
        answer1=True
    elif x%answer !=0:
        answer1=False

    return answer1

```

너무 간단하게 푼 풀이가 있어서 가져왔습니다
```ruby
def Harshad(n):
    return n % sum([int(c) for c in str(n)]) == 0
```
