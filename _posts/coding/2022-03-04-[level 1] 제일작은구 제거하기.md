##  python 풀이 

사실 처음에는 numpy로 sort했고 1문제가 시간때문에 오류로 떴다.
그런데 생각보다 쉬운 문제였다. 그냥 작은거 빼면 되니깐..
문제에서는 빈배열이면 -1로 리턴하라했는데 그런 조건문 안해도 왜 통과되는 지 의문..
```ruby
def solution(arr):
    answer=[]
    if  len(arr)==1:
	    answer.append(-1)

    arr.remove(min(arr))
    answer=arr
   
    return answer
```
