##  python 풀이 


```ruby
def solution(n, m):
    answer = []
    for i in range(min(n,m),0,-1):
        if (m%i==0)&(n%i==0):
            answer.append(i)
            break
    for j in range(max(n,m),(n*m)+1):
        if (j%n==0)&(j%m==0):
            answer.append(j)
            break
        
    return answer
```
